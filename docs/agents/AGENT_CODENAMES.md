## Usage in UI

Agents are displayed with their codenames in the Contigo Lab interface:
- **Agent Command Center** shows codenames with technical names in tooltips
- **Approval Queue** uses codenames for friendlier interactions
- **Chat @mentions** support both codenames and technical names

Example interactions:
> "@Sentinel validate this contract"
> "@Prospector find savings opportunities"
> "@Clockwork when does this expire?"
