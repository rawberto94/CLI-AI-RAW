# Savings Pipeline - User Guide

## Overview

Track and manage cost savings opportunities across your procurement portfolio. Monitor the pipeline from identification through realization.

**Access:** `/analytics/savings`

---

## Key Features

1. **Pipeline Overview** - Total weighted savings potential
2. **Opportunity Management** - Track opportunities by status
3. **Category Breakdown** - Savings by procurement category
4. **ROI Analysis** - Return on investment calculations
5. **Trend Tracking** - Historical savings performance

---

## Opportunity Statuses

- **Identified** - Opportunity discovered, not yet pursued
- **In Progress** - Actively working on realization
- **Realized** - Savings achieved and verified
- **Closed** - Opportunity closed (realized or abandoned)

---

## Quick Start

### View Pipeline
1. Navigate to `/analytics/savings`
2. Review pipeline overview cards
3. Check opportunities list
4. Filter by category or status

### Track Opportunity
1. Find opportunity in list
2. Note current status
3. Check probability and timeframe
4. Review potential savings

### Analyze by Category
1. Scroll to category breakdown
2. Review distribution
3. Identify focus areas
4. Plan initiatives

---

## Common Workflows

### Monthly Review
1. Check total pipeline value
2. Review in-progress opportunities
3. Verify realized savings
4. Update opportunity statuses

### Quarterly Planning
1. Analyze trends
2. Identify new opportunities
3. Prioritize by ROI
4. Allocate resources
